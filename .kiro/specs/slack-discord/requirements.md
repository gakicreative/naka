# Requirements Document

## Project Description (Input)
Integração com Slack e Discord para o SaaS Naka OS: alertas automáticos no canal configurado da equipe quando tarefas mudam de status, são atribuídas, vencem hoje ou recebem comentários. A organização configura a URL do webhook do Slack/Discord nas settings e escolhe quais eventos disparam notificações. Sem bot bidirecional nesta fase — apenas envio de mensagens.
