# Requirements Document

## Project Description (Input)
Sistema de webhooks para o SaaS Naka OS: organizações podem registrar URLs externas para receber notificações HTTP quando eventos ocorrem no sistema (tarefa criada, status mudou, contrato assinado, cliente adicionado, etc.). Inclui painel de gerenciamento de webhooks, log de entregas, retry automático em falhas e segurança via HMAC signature.
